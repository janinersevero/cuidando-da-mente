# Sons do Jogo Cuidando da Mente

Esta pasta deve conter os arquivos de som para o jogo. Os sons são opcionais - o jogo funcionará sem eles, mas a experiência será melhor com áudio.

## Arquivos de Som Necessários:

### Sons dos Elementos (ambiente - devem fazer loop):
- `fire.mp3` ou `fire.wav` - Som de fogo crepitando (suave)
- `water.mp3` ou `water.wav` - Som de água fluindo ou ondas suaves
- `wind.mp3` ou `wind.wav` - Som de vento suave
- `birds.mp3` ou `birds.wav` - Som de pássaros ou natureza (para Terra)

### Sons de Feedback (curtos):
- `success.mp3` ou `success.wav` - Som de sucesso/acerto
- `error.mp3` ou `error.wav` - Som de erro/tentativa (suave, não agressivo)

## Características Recomendadas:

- **Duração**: Sons ambiente: 30-60 segundos para loop; Sons de feedback: 1-3 segundos
- **Volume**: Baixo a moderado (será controlado pela aplicação)
- **Tom**: Relaxante, não estimulante
- **Formato**: MP3 preferível para compatibilidade

## Fontes Sugeridas:

- Freesound.org (Creative Commons)
- Zapsplat.com (com conta gratuita)
- YouTube Audio Library
- Pixabay Music

## Nota:

Se os arquivos não estiverem presentes, o jogo continuará funcionando normalmente sem áudio. Os erros de carregamento de áudio são tratados silenciosamente.
